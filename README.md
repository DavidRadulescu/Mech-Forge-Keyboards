# ⌨️ Mech Forge - Magazin Tastaturi Custom

Acest repository conține proiectul pentru laboratorul de **Tehnici Web**, reprezentând interfața (front-end) pentru un magazin online dedicat pasionaților de tastaturi mecanice custom.

## 📖 Descrierea Proiectului
Platforma **Mech Forge** permite utilizatorilor să achiziționeze atât tastaturi pre-asamblate gata de utilizare, cât și componente individuale pentru a-și construi propriile configurații de la zero. 

Dincolo de funcționalitățile clasice de e-commerce (filtrare avansată, coș de cumpărături, sistem de conturi), site-ul pune un accent puternic pe interactivitate. Elementul central de atracție este **"Typing Arena"**, o secțiune interactivă multiplayer unde utilizatorii se pot întrece în timp real la teste de viteză de tastare (WPM) pentru a câștiga reduceri.

---

### Categorii de Produse
* **Tastaturi Pre-asamblate:** Diferite layout-uri (60%, 65%, TKL, Full-Size) și destinații (Gaming, Office, Hibrid).
* **Componente Custom:**
  * *Switch-uri:* Liniare, Tactile, Clicky.
  * *Keycaps (Taste):* PBT, ABS, diverse profile (Cherry, OEM).
  * *Barebones (Kit-uri):* Carcase și PCB-uri.
* **Accesorii & Modding:** Cabluri custom, lubrifianți (Krytox), folii/films.

### Harta Site-ului (Pagini Principale)
1. **Acasă (Home):** Prezentare generală, Hero Banner, bestsellers.
2. **Magazin (Shop):** Grilă de produse cu panou lateral pentru filtre complexe.
3. **Pagină Produs:** Galerie foto, specificații tehnice detaliate.
4. **Typing Arena:** Interfață de joc multiplayer bazată pe WebSockets / Canvas.
5. **Cont / Panou Admin:** Istoric comenzi pentru useri, operațiuni CRUD pentru admini.


## 🔍 Analiza Concurenței
În faza de cercetare, au fost analizate următoarele platforme din nișă:
1. **QwertyKey:** Filtre vizuale excelente, dar o pagină principală prea aglomerată.
2. **KBDFans:** Calitate excepțională a prezentărilor 3D, însă un meniu de navigare copleșitor.
3. **Keychron:** Design curat și specificații tehnice foarte bine organizate; lipsa unui Dark Mode nativ.
4. **NovelKeys:** Aspect premium cu utilizare eficientă a spațiului negativ, dar filtrare tehnică limitată.
